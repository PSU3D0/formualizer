---

## 1) Install

```bash
pip install formualizer
```

## 2) Evaluate a formula

```python
import formualizer as fz

wb = fz.Workbook()
wb.add_sheet("Sheet1")

wb.set_value("Sheet1", 1, 1, 100)
wb.set_value("Sheet1", 2, 1, 20)
wb.set_formula("Sheet1", 1, 2, "=A1-A2")

result = wb.evaluate_cell("Sheet1", 1, 2)
print(result)  # 80.0
```
