This guide introduces the Python binding workflow for parsing and evaluating formulas.

## Steps

1. Install Formualizer Python package in your environment.
2. Import the binding and create a minimal evaluator context.
3. Evaluate a simple formula and print the result.
4. Add a small test to lock behavior.
