This quickstart will walk through adding Formualizer crates to a Rust workspace and evaluating your first formula.

## Steps

1. Add the relevant Formualizer crate(s) to `Cargo.toml`.
2. Create a minimal evaluator setup.
3. Parse and evaluate a simple formula.
4. Assert expected output in a test.
