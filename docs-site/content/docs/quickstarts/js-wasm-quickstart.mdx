This quickstart covers using the WASM package from a JavaScript project and evaluating formulas in the browser or Node.js.

## Steps

1. Install the WASM package and runtime dependencies.
2. Initialize the module in your JavaScript entry point.
3. Evaluate a formula and inspect the result.
4. Add a smoke test for loading and execution.
