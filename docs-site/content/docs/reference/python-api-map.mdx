> **v1 scaffold:** This page tracks the intended Python-facing API shape while bindings continue to evolve.

## Planned Surface Areas

- **Parser access:** Parse a formula string and inspect AST/diagnostics.
- **Evaluator access:** Evaluate formulas against provided context data.
- **Value conversion:** Convert between Python-native values and Formualizer values.
- **Error translation:** Preserve structured engine errors with Python-friendly exceptions.

## Mapping Notes

- Python parser/evaluator endpoints map onto Rust parser and eval crates.
- Function behavior should stay aligned with the shared function registry.
- Error payloads should remain compatible with [Errors and Types](/docs/reference/errors-and-types).
