title: "IMDIV"

---

## Summary

Divides the first complex value by the second.

## Remarks
 - Inputs are coerced to complex-number text before parsing.
 - Returns `#DIV/0!` when the divisor is `0+0i`.
 - Invalid complex text returns `#NUM!`.
