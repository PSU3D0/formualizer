---
title: "ERFC"
description: "Returns the complementary error function of a number."
---

## Summary

`ERFC(x)` is equivalent to `1 - ERF(x)`.

## Remarks
 - Input is numerically coerced.
 - Results are approximate floating-point values.
 - Propagates input errors.
