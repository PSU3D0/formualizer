## Summary

This is commonly used for security price formats such as thirty-seconds (`fraction = 32`).

## Remarks
 - `fraction` is truncated to an integer denominator and must be `&gt;= 1`.
 - Sign convention: sign is preserved (`-x` maps to `-result`).
 - No periodic rate is involved in this conversion.
 - Returns `#NUM!` when `fraction &lt; 1` after truncation.
 - Fractional parsing uses denominator digit width (`ceil(log10(fraction))`).

## FAQ

### Why does `DOLLARDE` truncate `fraction`?
The denominator is treated as an integer quote base; values below `1` after truncation return `#NUM!`.
