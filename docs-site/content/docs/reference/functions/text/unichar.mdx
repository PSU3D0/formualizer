## Remarks
 - Input is truncated to an integer code point.
 - Code point `0`, surrogate range, and values above `0x10FFFF` return `#VALUE!`.
 - Errors are propagated unchanged.
 - Non-numeric inputs are coerced with numeric coercion rules.
