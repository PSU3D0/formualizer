## Remarks
 - Standard mappings include: `#NULL!`=1, `#DIV/0!`=2, `#VALUE!`=3, `#REF!`=4, `#NAME?`=5, `#NUM!`=6, `#N/A`=7.
 - Non-error inputs return `#N/A`.
 - Additional internal error kinds may map to extended non-standard codes.
