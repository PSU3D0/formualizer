## Summary

`PERMUT` computes `n!/(n-k)!` after truncating both inputs toward zero.

## Remarks
 - Fractional inputs are truncated to integers.
 - If `n &lt; 0`, `k &lt; 0`, or `k &gt; n`, the function returns `#NUM!`.
 - Argument errors propagate directly.
