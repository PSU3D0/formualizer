---
title: "COUNTIF"
description: "Counts cells in a range that satisfy a single criterion."
---

## Summary

`COUNTIF` evaluates each candidate cell against one criteria expression.

## Remarks
 - Criteria support numeric comparisons and wildcard text matching.
 - Matching is case-insensitive for text criteria.
 - Non-matching or blank cells are not counted.

## Examples

<Tabs items={["Count numbers greater than 10", "Count text with wildcard", "Exact-match criterion"]}>
<Tab value="Count numbers greater than 10">
<FunctionSandbox
  title="Count numbers greater than 10"
  formula={`=COUNTIF(A1:A3, ">10")`}
  grid={{"A1":"5","A2":"15","A3":"22"}}

</Tab>

<FunctionSandbox

<Tab value="Exact-match criterion">
<FunctionSandbox
