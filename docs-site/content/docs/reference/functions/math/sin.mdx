## Summary

Returns the sine of an angle in radians.

## Remarks
 - Input is interpreted as radians, not degrees.
 - Supports scalar and array-style elementwise evaluation.

## Examples

<Tabs items={["Sine of PI/2", "Sine from a cell value"]}>
<Tab value="Sine of PI/2">
<FunctionSandbox
  title="Sine of PI/2"
  formula={`=SIN(PI()/2)`}
  grid={{}}
  expected={"1"}
/>
</Tab>

<Tab value="Sine from a cell value">
<FunctionSandbox
  title="Sine from a cell value"
  formula={`=SIN(A1)`}
  grid={{"A1":"0"}}
  expected={"0"}
/>
</Tab>
