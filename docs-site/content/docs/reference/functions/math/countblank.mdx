---
title: "COUNTBLANK"
description: "Counts blank cells, including empty-string text results."
---

## Summary

`COUNTBLANK` treats both true empty cells and `""` text values as blank.

## Remarks
 - Empty-string text values are counted.
 - Numbers, booleans, and non-empty text are not counted.
 - Supports scalar arguments and ranges.

## Examples

<Tabs items={["Count blanks in a range", "Scalar empty-string counts as blank", "Non-empty values are excluded"]}>
<Tab value="Count blanks in a range">
<FunctionSandbox
  title="Count blanks in a range"

  grid={{"A1":"10","A2":""}}

/>
</Tab>

<Tab value="Scalar empty-string counts as blank">
<FunctionSandbox

/>
</Tab>

<Tab value="Non-empty values are excluded">
<FunctionSandbox
  title="Non-empty values are excluded"

</Tab>

</Tabs>
