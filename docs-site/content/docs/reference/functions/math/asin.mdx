## Summary

Use `ASIN` to recover an angle from a normalized ratio.

## Remarks
 - Domain: input must be between `-1` and `1`, inclusive.
 - Radians: output is in the range `[-PI()/2, PI()/2]`.
 - Errors: returns `#NUM!` when the input is outside the valid domain.
