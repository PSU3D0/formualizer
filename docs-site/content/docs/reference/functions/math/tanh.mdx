## Summary

`TANH` computes `SINH(x) / COSH(x)`.

## Remarks
 - Domain: accepts any real number.
 - Radians: this function is hyperbolic, so the input is not treated as an angle in radians.
 - Errors: no function-specific domain errors are produced.
