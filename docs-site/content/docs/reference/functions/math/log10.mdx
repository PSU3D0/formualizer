## Summary

Returns the base-10 logarithm of a positive number.

## Remarks
 - `number` must be greater than `0`; otherwise the function returns `#NUM!`.
 - `LOG10(POWER(10,x))` returns `x` up to floating-point precision.
 - Input errors are propagated.
