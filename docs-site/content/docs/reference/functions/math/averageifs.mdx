---
title: "AVERAGEIFS"
description: "Returns the average of cells that satisfy multiple criteria."
---

## Summary

`AVERAGEIFS` filters by all criteria pairs, then averages matching numeric values.

## Remarks
 - The first argument is the average target range.
 - Criteria are supplied in `(criteria_range, criteria)` pairs.
 - If no numeric cells match, the function returns `#DIV/0!`.

## Examples

<Tabs items={["Average with two criteria", "Average over inline arrays", "No matches returns divide-by-zero"]}>
<Tab value="Average with two criteria">
