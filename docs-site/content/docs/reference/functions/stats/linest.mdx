## Summary

LINEST(known_y's, known_x's, [const], [stats]) - Returns statistics describing the linear trend With stats=FALSE (default): returns 1x2 array [slope, intercept] With stats=TRUE: returns 5x2 array with regression statistics

## Formula example

```text
=LINEST(A1)
```
