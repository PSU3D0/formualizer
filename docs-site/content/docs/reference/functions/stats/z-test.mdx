## Summary

Z.TEST(array, x, [sigma]) - Returns the one-tailed P-value of a z-test. z = (mean(array) - x) / (sigma / sqrt(n)) Returns 1 - NORM.S.DIST(z, TRUE) If sigma is omitted, uses the population standard deviation of the array.

## Formula example

```text
=Z.TEST(A1, B1)
```
