## Summary

TREND(known_y's, [known_x's], [new_x's], [const]) - Returns y values along a linear trend Uses linear regression y = mx + b - If new_x's provided, calculates trend values for those x's - If new_x's omitted, uses known_x's - const=TRUE (default): calculate intercept normally - const=FALSE: force intercept through origin

## Formula example

```text
=TREND(A1)
```
