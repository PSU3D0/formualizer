## Summary

LOGEST(known_y's, [known_x's], [const], [stats]) - Returns parameters of exponential curve Returns array: [[m, b]] when stats=FALSE Returns 5x2 array with statistics when stats=TRUE (like LINEST) The exponential curve is y = b * m^x

## Formula example

```text
=LOGEST(A1)
```
