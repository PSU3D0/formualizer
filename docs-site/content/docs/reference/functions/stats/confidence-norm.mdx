## Summary

Returns the confidence interval for a population mean using a normal distribution. Formula: z_crit * standard_dev / sqrt(size), where z_crit = NORM.S.INV(1 - alpha/2)

## Formula example

```text
=CONFIDENCE.NORM(A1, B1)
```
