## Summary

GROWTH(known_y's, [known_x's], [new_x's], [const]) - Returns values along exponential growth trend Uses exponential regression y = b * m^x - Similar parameters to TREND but for exponential growth - const=TRUE: calculate b normally - const=FALSE: force b = 1

## Formula example

```text
=GROWTH(A1)
```
