## Summary

T.DIST.2T(x, deg_freedom) - Returns the two-tailed Student's t-distribution Returns P(|T| &gt; x) = 2 * (1 - t_cdf(|x|, df))

## Formula example

```text
=T.DIST.2T(A1, B1)
```
