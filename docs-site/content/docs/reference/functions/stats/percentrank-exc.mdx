## Summary

PERCENTRANK.EXC(array, x, [significance]) - Returns percentage rank (exclusive) Same as PERCENTRANK.INC but excludes 0 and 1 from range Range is 1/(n+1) to n/(n+1)

## Formula example

```text
=PERCENTRANK.EXC(A1, B1)
```
