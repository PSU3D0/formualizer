## Summary

T.INV.2T(probability, deg_freedom) - Returns the two-tailed inverse of Student's t-distribution Returns the value t such that P(|T| &gt; t) = probability This is equivalent to t_inv(1 - probability/2, df)

## Formula example

```text
=T.INV.2T(A1, B1)
```
