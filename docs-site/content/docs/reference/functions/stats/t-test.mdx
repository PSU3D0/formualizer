## Summary

T.TEST(array1, array2, tails, type) - Returns probability for Student's t-test tails: 1 for one-tailed, 2 for two-tailed type: 1=paired, 2=two-sample equal variance, 3=two-sample unequal variance (Welch's)

## Formula example

```text
=T.TEST(A1, B1)
```
