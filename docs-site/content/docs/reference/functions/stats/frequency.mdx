## Summary

FREQUENCY(data_array, bins_array) - Returns frequency distribution Returns vertical array of frequencies Counts values in each bin: &lt;= bin[0], (bin[0], bin[1]], ..., &gt; bin[n-1] Returns array with one more element than bins_array

## Formula example

```text
=FREQUENCY(A1, B1)
```
