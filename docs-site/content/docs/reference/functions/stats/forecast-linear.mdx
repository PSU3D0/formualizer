## Summary

FORECAST.LINEAR(x, known_y's, known_x's) - Returns predicted y value for x using linear regression The formula is: y = intercept + slope * x where slope = sum((xi - mean_x)(yi - mean_y)) / sum((xi - mean_x)^2) and intercept = mean_y - slope * mean_x

## Formula example

```text
=FORECAST.LINEAR(A1, B1)
```
