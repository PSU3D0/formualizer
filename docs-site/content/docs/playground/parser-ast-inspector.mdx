> **v1 scaffold:** This page outlines the initial parser inspection workflow.

## Intended v1 Capabilities

- Parse formulas and display parser diagnostics with source positions.
- Inspect AST structure in a readable tree/JSON view.
- Compare parse output before and after expression edits.

## Planned Follow-ups

- Toggle between compact and expanded AST views.
- Optional token stream pane for parser debugging.
