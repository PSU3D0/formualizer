Formualizer is an embeddable spreadsheet formula engine for apps, services, and automation pipelines.

## Path by persona

- **Rust app/service developer**: Start with [Rust Quickstart](/docs/quickstarts/rust-quickstart), then [First Workbook in 5 Minutes](/docs/quickstarts/first-workbook-in-5-minutes), then [Custom Functions (Rust/Python/JS)](/docs/guides/custom-functions-rust-python-js).
- **Python automation/data engineer**: Start with [Python Quickstart](/docs/quickstarts/python-quickstart), then [First Workbook in 5 Minutes](/docs/quickstarts/first-workbook-in-5-minutes), then [Workbook Edits and Batching](/docs/guides/workbook-edits-and-batching).
- **Web/Node engineer (WASM)**: Start with [JavaScript/WASM Quickstart](/docs/quickstarts/js-wasm-quickstart), then [First Workbook in 5 Minutes](/docs/quickstarts/first-workbook-in-5-minutes), then [WASM Plugins: Inspect, Attach, Bind](/docs/guides/wasm-plugins-inspect-attach-bind).
- **Engine contributor**: Start with [Architecture at a Glance](/docs/introduction/architecture-at-a-glance), then [Core Concepts](/docs/core-concepts), then [Deterministic Testing](/docs/guides/deterministic-testing).
