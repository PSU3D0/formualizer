---
title: What is Formualizer?
description: A concise definition of the Formualizer project and its scope.
---

Formualizer is an engine and toolkit for parsing and evaluating spreadsheet-style formulas across multiple runtimes.

At a high level, it provides:

- A parser for turning formula text into structured representations.
- An evaluation engine for computing results.
- Bindings for environments like Rust, Python, and WebAssembly.

## Who It Is For

Formualizer is useful for teams building spreadsheet-like features, data automation flows, and cross-platform formula services.
